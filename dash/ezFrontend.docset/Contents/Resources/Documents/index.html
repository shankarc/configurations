Build EzReverse Proxy Thrift

mvn generate-resources -P gen-thrift

Generating the thrift code

Run this command on host machine.

Install generated ezbake-reverseproxy-thrift library

sudo 'which pip' install .

Update ezbake-reverseproxy-thrift module in vagrant

In vagrant: After changing to /ezbake-reverseproxy-thrift/src/main/python run this command. The setup.py has the version number of the module Note: Make sure to uninstall the previous version.

sudo `which pip` uninstall -y ezbake-reverseproxy-thrift
